# ZPI-IO8R

ZPI-IO8R 최신 배포 자료를 넣는 폴더입니다. 이 폴더 루트의 파일이 외부 다운로드 링크의 기준입니다.

## 현재 배포

```text
version: v3.0
updated: 2026-05-21
```

## 파일

```text
manual.pdf
manual-image.webp
schematic.png
release.json
```

외부 링크를 유지하려면 같은 종류의 자료는 같은 파일명으로 교체합니다.

## 보관

이전 자료를 남길 때는 `_archive/YYYY-MM-DD_vX.Y.Z/` 형식의 하위 폴더를 만들고 그 안에 보관합니다.
