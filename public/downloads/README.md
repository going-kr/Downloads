# Downloads

다운로드 URL의 기준 폴더입니다. 제품별 최신 자료는 각 제품 폴더 루트에 직접 넣습니다.

## 폴더 규칙

```text
senvas/                 Senvas 최신 자료
zpi-io8r/               ZPI-IO8R 최신 자료
senbrix/                Senbrix 최신 자료
bundles/                여러 제품을 묶은 패키지 자료
```

새 제품을 추가할 때는 제품명을 소문자 슬러그로 만든 폴더를 추가합니다.

```text
제품명: Sen Example
폴더명: sen-example
```
